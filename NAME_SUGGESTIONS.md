# Extension Name Suggestions

## Current Name: **CleanRead**
- ✅ Clean, professional
- ❌ Not SEO-friendly (people don't search "clean read")
- ❌ Doesn't clearly indicate it's an ad blocker

---

## Naming Strategy Analysis

### What People Actually Search For:
- "ad blocker" (most common)
- "free ad blocker"
- "lightweight ad blocker"
- "simple ad blocker"
- "fast ad blocker"

### Competitive Differentiation:
**Existing:** "Ad Remover - Ad Blocker for Chrome"
- Long, descriptive
- Enterprise-feeling

**Your Advantage:**
- Lightweight
- Free
- Simple
- Fast

**Suggested Format:**
`[ShortName] - [Key Benefit] Ad Blocker`

Examples:
- `Lite AdBlock - Free & Fast`
- `TinyBlock - Lightweight Ad Blocker`
- `SwiftBlock - Free Sidebar Ad Blocker`

---

## Final Recommendations

### For Maximum Discoverability:
```
Name: Lite AdBlock
Description: Lightweight, privacy-first ad blocker. Automatically removes sidebar ads and common advertising elements.
```

### For Brand Recognition:
```
Name: AdSweep Lite
Description: Fast, free ad blocker. Sweep away sidebar ads instantly with minimal resource usage.
```

### For Niche Appeal (sidebar-specific):
```
Name: SidebarBlock
Description: Specialized lightweight blocker for sidebar ads. Free, fast, and privacy-focused.
```

---

## To Change the Name:

Update `manifest.json`:
```json
{
  "name": "Lite AdBlock",
  "short_name": "LiteAdBlock",
  "description": "Lightweight, privacy-first ad blocker. Automatically removes sidebar ads and common advertising elements."
}
```
