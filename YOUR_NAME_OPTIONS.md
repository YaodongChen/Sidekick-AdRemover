## Quick Switch Guide

### To Switch to "Sidebar Purify":

1. Edit `manifest.json`:
```json
{
  "name": "Sidebar Purify",
  "short_name": "Purify",
  "description": "Free sidebar ad blocker. No setup required - just install and browse. Lightweight, privacy-first, zero configuration."
}
```

2. Edit `src/popup/popup.html`:
```html
<h3>Purify</h3>
<p>Clean Sidebar Ads</p>
```

3. Rebuild:
```bash
npm run build
```

---
